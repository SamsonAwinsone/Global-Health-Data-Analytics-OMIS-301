# Global-Health-Data-Analytics-OMIS-301
Focuses on the application of business analytics techniques to a global health dataset. Developed as part of the OMIS 301 course, the project integrates theoretical data classification with practical data science workflows using Python to derive insights into disease prevalence, healthcare access, and treatment outcomes across various countries.



Key Objectives

Data Exploration & EDA: Performing Exploratory Data Analysis (EDA) on a health dataset featuring 22 variables, including prevalence rates, mortality rates, and healthcare infrastructure metrics (doctors and hospital beds per 1000).  


Statistical Analysis: Generating descriptive statistics to identify trends in disease categories such as Infectious, Non-Communicable, and Genetic conditions.  


Data Classification: Categorizing variables into nominal, ordinal, interval, and ratio types to determine appropriate analytical models.  


Predictive Modeling Framework: Identifying key predictors for recovery rates and establishing a supervised learning framework to forecast health outcomes.  


Technologies Used

Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn   

Tools: Jupyter Notebook

Dataset Features
The analysis utilizes a comprehensive dataset covering:


Demographics: Age groups, gender, and urbanization rates.  


Health Metrics: Incidence rates, DALYs (Disability-Adjusted Life Years), and recovery rates.  


Economics: Per capita income and average treatment costs.  


Infrastructure: Healthcare access percentages and education indices.
